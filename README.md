# U1 - Proyecto Integrador

## Descripción del proyecto

Este proyecto consiste en la creación de un escenario procedural en Blender, basado en la actividad "Escenario Procedural".

Además, se implementó una animación donde la cámara recorre el escenario a través de un camino utilizando una curva y la restricción Follow Path.

---

## Objetivo

- Generar un escenario mediante modelado procedural.
- Implementar una animación de cámara que siga un recorrido.
- Renderizar el recorrido en formato de video.
- Documentar el proceso en GitHub.

---

## Contenido del repositorio

- Carpeta **blender/** → Archivo del proyecto (.blend)
- Carpeta **renders/** → Video de la animación renderizada

---

## Animación de cámara

Se utilizó una curva como trayectoria y se aplicó a la cámara la restricción **Follow Path**.  
Se animó el recorrido usando keyframes desde el frame 1 hasta el frame 250.

---

## Motor de render

Se utilizó el motor Eevee para realizar el render de la animación en formato MP4 (H.264).
